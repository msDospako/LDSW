Conclusiones Unidad 1

Intentar ver o diseñar una red de comunicaciones como una autopista digital que conecta nuestros aparatos para que puedan compartir información. darnos cuenta que necesitamos aparatos para hablar, carreteras para viajar, organizadores de tráfico para dirigir y reglas para entenderse ademas de que todos estos componentes trabajan juntos para que podamos ver videos, enviar mensajes y hacer muchas otras cosas en el mundo conectado de hoy es bastante divertido de imaginar.

Unidad 2
Conclusiones Unidad 2

Entender las "reglas del juego" (Protocolos de Comunicación) o llevarlo a imaginar que cada idioma es un protocolo. Para que dos aparatos se entiendan, deben hablar el mismo "idioma". Los protocolos son las reglas que definen cómo se formatea, se envía y se recibe la información para que todos se entiendan. Por ejemplo, el "idioma" que usan las páginas web se llama HTTP. y solo lo entienden de esta manera y a su vez interactuar con clientes y servidores y darte cuenta de manera practica como montar un home server y poder resguardar bibliotecas completas de musica, archivos o diversas cosa

Unidad 3
Conclusiones Unidad 3

Pensar en la web como un gran espacio digital que usamos para casi todo. Los diferentes tipos de sitios web son como los diferentes tipos de "locales" que encontramos en una plaza, cada uno con su propio propósito y forma de interactuar con nosotros. Entender esta variedad nos ayuda a navegar mejor en internet y a aprovechar al máximo lo que nos ofrece. Es como conocer los diferentes tipos de negocios que hay en el centro de Guadalajara para saber a dónde ir cuando necesitamos algo específico.

Conclusiones PI

Entender los fundamentos de Internet nos da una visión clara de cómo funciona el mundo digital que usamos a diario. No es solo una herramienta mágica, sino un sistema complejo y organizado que se basa en infraestructura física, reglas de comunicación y principios de seguridad. Con este conocimiento, podemos ser usuarios más informados, responsables y preparados para el futuro y más allá de ver esto como un entrar y salir darle su peso como la gran herramienta que es entender lo que podemos encontrar y como lo podemos encontrar
